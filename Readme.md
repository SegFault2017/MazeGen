## Maze Generator and Path Finding
Generate an n x n maze and find a path using A* algorithm.

## Constructor Methods

1. Recursive Backtracker(Default)
2. Kruskal
3. Prim



## Solver Methods

1. BFS
2. DFS
3. A* by Manhanttan
